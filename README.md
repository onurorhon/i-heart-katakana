# I♥︎Katakana

A focused iOS app for practicing katakana word recognition with real Japanese loanwords.

Many apps help learners memorize katakana characters. Far fewer help them practice reading real katakana words as they appear in everyday Japanese. I♥︎Katakana fills that gap with curated content and a clean, distraction-free experience.
